# 📚 React Book Finder App

A responsive React web application that allows users to search for books using the Open Library API.  
Users can view book results, browse pages, and click on a book to see detailed information in a modal.

## 🌍 Live Demo

👉 https://react-book-finder-app-alpha.vercel.app/

---

## ✨ Features

- 🔍 Search books by title
- 📖 View book results with cover images
- 📄 Pagination (Previous / Next)
- 🪄 Click to view detailed book information
- 📚 View description and subjects
- 🎨 Modern responsive UI
- ⚡ Fast API integration
- 🚀 Deployed on Vercel

---

## 🛠️ Technologies Used

- React.js (Functional Components)
- React Hooks (useState, useEffect)
- JavaScript (ES6+)
- CSS3
- Open Library API
- Vercel (Deployment)
- GitHub (Version Control)

---

## 📡 API Used

This project uses the **Open Library API**:

- Search API: https://openlibrary.org/search.json
- Book Details API: https://openlibrary.org/{book.key}.json
- Cover Images: https://covers.openlibrary.org/

---

## 🚀 Installation & Setup

To run this project locally:

- git clone https://github.com/vyshnavi2306/react-book-finder-app.git
- cd react-book-finder-app
- npm install
- npm start

The app will run on: http://localhost:3000

---

## 🏗️ Build for Production

- npm run build

## 🌐 Deployment

This project is deployed using Vercel with automatic deployment from GitHub.
